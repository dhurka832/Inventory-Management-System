# Inventory Management System

A **Python console-based Inventory Management System** with **MySQL database integration** and **PrettyTable** for tabular display. This system allows users to manage categories, products, invoices, and low-stock alerts efficiently.

---

## **Features**

- Add and view **categories**.
- Add, view, update, and delete **products**.
- Generate **invoices** for sold products.
- Display **low-stock alerts**.
- Console output formatted using **PrettyTable**.

---

## **Requirements**

- Python
- MySQL Server / MySQL Workbench  
- Python packages:
  - `mysql-connector-python`
  - `prettytable`
